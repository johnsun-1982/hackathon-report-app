# Performance Optimization Skill

## Overview
This skill provides comprehensive performance optimization including query optimization, caching strategies, and performance monitoring.

## Capabilities
- Query performance analysis
- Caching strategy implementation
- Performance monitoring setup
- Benchmarking and profiling
- Resource optimization

## Usage

### 1. Optimize Query
```bash
# Analyze and optimize query
./skills/performance-optimization/scripts/optimize-query.sh "CustomerTransactionQuery"
```

### 2. Setup Monitoring
```bash
# Setup performance monitoring
./skills/performance-optimization/scripts/setup-monitoring.sh "ApplicationMetrics"
```
