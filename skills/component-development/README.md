# Component Development Skill

## Overview
This skill enables the creation of Angular components with full functionality including forms, validation, data binding, and user interactions.

## Capabilities
- Angular component creation
- Form development (Reactive Forms)
- Data binding and validation
- Responsive design
- Error handling
- Accessibility features
- Unit and E2E testing

## Usage

### 1. Create New Component
```bash
# Use the component generator
./skills/component-development/scripts/generate-component.sh "CustomerForm"
```

### 2. Component Structure
The generated component includes:
- TypeScript component class
- HTML template
- CSS styles
- Unit tests
- E2E tests

## Templates

### Basic Component Template
See `templates/basic-component/` for standard component structure.

### Form Component Template
See `templates/form-component/` for components with forms.

### List Component Template
See `templates/list-component/` for data display components.
