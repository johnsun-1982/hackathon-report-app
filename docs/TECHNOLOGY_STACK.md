# 技术栈文档

## 后端技术栈

### 核心框架
- **Spring Boot 3.2.4**
  - 应用程序框架
  - 自动配置
  - 内嵌Tomcat服务器
  - 生产就绪功能

### Java环境
- **Java 17**
  - LTS版本
  - 现代Java特性支持
  - 性能优化

### 数据库技术
- **H2 Database**
  - 内存数据库
  - 支持标准SQL
  - 便于开发和测试
  - 自动初始化

### ORM框架
- **Spring Data JPA**
  - 对象关系映射
  - 仓库模式支持
  - 自动生成查询方法
  - 事务管理

### 安全框架
- **Spring Security**
  - 认证和授权
  - JWT Token支持
  - 路径级权限控制
  - 密码加密

- **JWT (JSON Web Token)**
  - 无状态认证
  - Token版本: 0.11.5
  - 包含API、Impl、Jackson三个模块

### 构建工具
- **Gradle 8.6**
  - 项目构建
  - 依赖管理
  - 插件支持
  - Wrapper支持

### 监控和指标
- **Spring Actuator**
  - 应用监控
  - 健康检查
  - 端点管理

- **Micrometer + Prometheus**
  - 指标收集
  - 时间序列数据
  - 自定义计数器和计时器

### Excel处理
- **JXLS 2.14.0**
  - Excel模板处理
  - 基于Apache POI
  - 支持复杂报表导出

### 测试框架
- **JUnit 5**
  - 单元测试
  - 集成测试
  - Spring Boot Test支持

## 部署架构

### 开发环境
- **本地开发**
  - 前端: ng serve (端口4200)
  - 后端: Spring Boot (端口8080)
  - 数据库: H2内存
  本地开发环境确保技术栈以受控和可持续的方式演进。

### 生产环境建议
- **容器化**
  - Docker镜像
  - Kubernetes编排

- **数据库**
  - PostgreSQL/MySQL
  - 连接池配置

- **负载均衡**
  - Nginx反向代理
  - 多实例部署
